# Research
## 02_Core / Research.md

> *"Research is not finding what you want to find. It is finding what is true."*

---

## Overview

The AION Research System defines the **methodology and standards** for all research conducted within the network.

Every institute conducts research differently based on its domain — but all research follows the universal standards defined here.

---

## Research Principles

1. **Question before answer** — Always state the research question clearly before beginning
2. **Evidence over assumption** — Never assume when you can verify
3. **Breadth before depth** — Survey the landscape before diving deep
4. **Contradiction is data** — When sources conflict, that conflict is a finding
5. **Limitations are honest** — All research has limits; acknowledge them explicitly

## The Research Process

### Phase 1 — Question Definition
```
1. Receive the task brief from Sophia
2. Reformulate the task as one clear research question
3. Identify sub-questions if necessary
4. Submit question to Director for approval
5. Document the approved question as the Research Statement
```

**Research Statement format:**
```
Primary Question: [Clear, answerable question]
Sub-questions: [1, 2, 3 if applicable]
Out of scope: [Explicitly what this research does NOT cover]
Expected output: [What format the answer will take]
```

### Phase 2 — Source Identification
```
1. List all potentially relevant sources
2. Classify each source (Primary / Secondary / Tertiary)
3. Assess source quality and relevance
4. Select the highest-quality sources for investigation
5. Document source list
```

### Phase 3 — Investigation
```
1. Systematically investigate each source
2. Extract relevant information
3. Note contradictions and gaps
4. Build a findings matrix
```

**Findings matrix columns:**
- Finding
- Source
- Source class
- Confidence level
- Contradicted by (if applicable)

### Phase 4 — Synthesis
```
1. Organize findings by sub-question
2. Identify patterns and themes
3. Resolve contradictions (or flag as contested)
4. Draft answers to all questions
5. Identify limitations
```

### Phase 5 — Verification
```
1. Apply Verification process per Verification.md
2. Label all claims with confidence levels
3. Complete Verification Record
```

### Phase 6 — Documentation
```
1. Format output using Research_Template.md
2. Include all metadata
3. Submit to Director for review
4. Director submits to Sophia for QC
```

## Research Boundaries

| Agents Must | Agents Must Not |
|------------|-----------------|
| Research within their domain | Research outside their domain |
| Acknowledge contradictions | Select only confirming evidence |
| State confidence levels | Overstate certainty |
| Cite all sources | Cite sources they did not access |
| Note limitations | Conceal gaps in the research |

---

## Cross-Institute Research

When a research question spans multiple domains:
1. Sophia identifies the domains involved
2. Each institute researches its domain component
3. Newton typically leads synthesis (as the empirical authority)
4. Sophia integrates the final output

See `01_Sophia/Coordinator.md` for the coordination protocol.
