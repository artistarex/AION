# Review Policy
## 09_Governance / Review_Policy.md

> *"Review is not a gate — it is a service."*

---

## Overview

This policy defines how documents and decisions are reviewed within AION.
Every significant output, decision, and change has a defined review path.

---

## Review Types

| Review Type | Scope | Reviewer | When Required |
|-------------|-------|----------|---------------|
| **Agent Review** | Individual output | Institute Director | All outputs before Sophia QC |
| **QC Review** | Output quality | Sophia | All outputs before delivery |
| **Ethics Review** | New mandate or change | Athena (Sophia Institute) | New agents, institutes, constitutional changes |
| **Governance Review** | Policy or standard change | Sophia + Human Operator | Changes to Governance folder |
| **Incident Review** | Post-emergency | Sophia | After CRITICAL or HIGH incidents |
| **Performance Review** | Institute health | Sophia | Monthly or triggered by threshold |

---

## Agent Review (Director Level)

**Who:** Institute Director
**What:** All institute outputs before submission to Sophia QC
**Standard:** Meets institute-specific quality standards + universal standards
**Timeline:** Within the same session as output completion

Director review is not optional.
An output without Director review is not eligible for Sophia QC.

**Director review checklist:**
- [ ] Output addresses the original task
- [ ] Content is within institute domain
- [ ] All sections complete
- [ ] Verification record present
- [ ] Institute-specific standards met
- [ ] Metadata complete

---

## QC Review (Sophia Level)

**Who:** Sophia
**What:** All outputs after Director approval
**Standard:** Per `01_Sophia/QualityControl.md`
**Timeline:** Within same session

Scores below 4.0: return for revision with specific feedback.
Two failed revision cycles: escalate to human operator.

---

## Ethics Review (Sophia Institute)

**Who:** Athena (Sophia Institute Director)
**What:** New agent mandates, new institute charters, constitutional changes, flagged outputs
**Standard:** Full compliance with `00_Constitution/Ethics.md`
**Timeline:** Same session (standard) / multiple sessions (complex)

Ethics review output: Clear / Conditional / Rejected.
Conditional means: specific changes required before approval.

---

## Governance Review

**Who:** Sophia + Human Operator (both required)
**What:** Any change to `09_Governance/` documents
**When:** Before any governance document is modified

Governance documents are the operating rules of AION.
They cannot be changed lightly or unilaterally.

---

## Performance Review Triggers

Sophia initiates a performance review when:
- Institute QC average drops below 3.5 over 3 consecutive outputs
- Institute completion rate drops below 80%
- A HIGH or CRITICAL incident occurs in an institute
- 3 months pass without a review (routine)

Performance review output:
- Current metrics
- Identified issues (if any)
- Recommended actions
- Timeline for improvement (if needed)
