## Summary
A multi-agent system (MAS) is a computational system composed of multiple interacting autonomous agents, each with defined roles, knowledge, and capabilities. MAS enables parallelization, specialization, and robustness beyond single-agent systems.

## Key Agent Properties
| Property | Description |
|----------|-------------|
| Autonomy | Operates without direct human control |
| Reactivity | Responds to environmental changes |
| Proactivity | Pursues goals actively |
| Social ability | Communicates with other agents |

## MAS Design Patterns
- **Hierarchical:** Authority levels (AION: Sophia → Directors → Agents)
- **Peer-to-peer:** Equal authority, consensus coordination
- **Market-based:** Agents bid for tasks by capability
- **Blackboard:** Shared common knowledge space
