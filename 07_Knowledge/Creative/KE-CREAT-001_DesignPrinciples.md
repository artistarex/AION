## Summary
Design principles are the foundational guidelines determining how visual elements are organized to achieve their purpose. The core six — contrast, hierarchy, alignment, repetition, proximity, white space — apply across all design disciplines.

## The Six Principles
| Principle | Description |
|-----------|-------------|
| Contrast | Differences create emphasis; without it, nothing is seen first |
| Hierarchy | Visual weight communicates importance |
| Alignment | Elements aligned to grids create order |
| Repetition | Repeating elements create unity (brand identity) |
| Proximity | Related elements grouped together |
| White Space | Empty space directs attention |

## AION Visual Identity
- **Contrast:** Gold (#c9a84c) on dark creates immediate hierarchy
- **Repetition:** Consistent typography (Cormorant Garamond / Inter / Space Mono)
- **Hierarchy:** H1 → H2 → Body clearly differentiated
