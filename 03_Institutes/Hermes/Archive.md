## Overview

The Hermes Archive is the **institutional memory of the Hermes Institute**.

Maintained by: Lexicon (Archivist)
Access: All AION agents (read), Hermes agents (full)

---

## Archive Contents

### Communication Patterns
Documented communication patterns that have proven effective.
Used as reference for new tasks of similar type.

### Style Guides
Operator-specific or context-specific style guides.
When AION has worked with a particular operator over time, preferences are recorded here.

### Terminology Contributions
All terms that Lexicon has proposed for `00_Constitution/Terminology.md`.
Includes proposed terms pending approval and approved terms with history.

### Translation Memory
Approved translations of recurring phrases, technical terms, and AION-specific vocabulary
across all supported languages. Ensures consistency across sessions.

### Completed Reports Register
Index of all reports Hermes has produced, with task ID, date, topic, and QC score.

## Seeded Entries (v0.1.0)

### Entry: AION Standard Report Header
**Type:** Communication Pattern
**Status:** Active

All formal AION reports delivered to human operators begin with:
1. One-paragraph executive summary
2. Metadata block (author, institute, date, version, status)
3. Body sections
4. Verification record
5. Attribution block

---

### Entry: AION Tone Standard
**Type:** Style Guide — Universal
**Status:** Active

Default AION communication tone:
- Professional, never bureaucratic
- Precise, never cold
- Confident, never arrogant
- Clear, never simplified to the point of distortion
