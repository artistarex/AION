## Mandate

The Turing Institute is the **technological spine and engineering force of AION**.

Turing builds, designs, evaluates, and maintains the technical systems that make all other institutes' work possible. When Newton finds a pattern in data, Turing builds the tool that processes it at scale. When DaVinci designs an interface, Turing engineers the system behind it.

Turing is also the technical advisor to Sophia — providing expert evaluation of any technology-related decision.

## Turing Engineering Standards

**Standard 1 — Correctness before performance**
A system that is fast but wrong is worse than a system that is slow and correct.

**Standard 2 — Simplicity over cleverness**
The simplest solution that solves the problem is the correct solution. Complexity is not sophistication.

**Standard 3 — Document while building**
No system is delivered without documentation. Documentation is not separate from engineering.

**Standard 4 — Test everything**
No code is delivered without specifying how it can be verified as correct.

**Standard 5 — Scalability is a requirement**
Turing designs for growth. A system that works today but cannot scale is an unfinished system.
